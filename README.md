# Objetivo de sistema

Desarrollar una versión funcional del videojuego Atari Pong utilizando el lenguaje de progrmación Python y la biblioteca Tkinter, aplicando los conocimientos adquiridos durante la asignatura, como variables, operadores, estructuras condicionales, bucles, funciones y eventos det teclado. El sistema busca desmostrar la aplicación práctica de la lógica de programación mediante un juego interactivo.

# Descripción de funcionalidades

El sistema desarrollado cuenta con las siguientes funcionalidades:
- Interfáz gráfica creada en la bilioteca Tkinter.
- Control de dos jugadores mediante el teclado.
- Movimiento automático de la pelota.
- Detección de colisiones entre la pelota y las paletas.
- Rebote de la pelota en los bordes superior e inferior de la ventana.
- Registro y actualización del marcador de puntos.
- Reinicio automático de la pelota después de cada anotación.
- Finalización de la partida cuando un jugador alcanza cinco puntos.
- Código organizado en funciones para facilitar su comprensión y mantenimiento.
