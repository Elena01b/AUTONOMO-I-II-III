# Objetivo de sistema

Desarrollar una versión funcional del videojuego Atari Pong utilizando el lenguaje de progrmación Python y la biblioteca Tkinter, aplicando los conocimientos adquiridos durante la asignatura, como variables, operadores, estructuras condicionales, bucles, funciones y eventos det teclado. El sistema busca desmostrar la aplicación práctica de la lógica de programación mediante un juego interactivo.

# Descripción de funcionalidades

El sistema desarrollado cuenta con las siguientes funcionalidades:
- Interfáz gráfica creada en la bilioteca Tkinter.
- Control de dos jugadores mediante el teclado.
- Movimiento automático de la pelota.
- Detección de colisiones entre la pelota y las paletas.
- Rebote de la pelota en los bordes superior e inferior de la ventana.
- Registro y actualización del marcador de puntos.
- Reinicio automático de la pelota después de cada anotación.
- Finalización de la partida cuando un jugador alcanza cinco puntos.
- Código organizado en funciones para facilitar su comprensión y mantenimiento.

# Explicación del sistema desarrollado

El sistema desarrollado corresponde a una versión funcional del juego Atari Pong.
Al iniciar la aplicación, se muestra una ventana con una cancha de juego, dos paletas y una pelota. Cada jugador controla una paleta utilizando el teclado: el jugador izquierdo emplea las teclas w y s, mientras que el jugador derecho utiliza las flechas arriba y abajo.
La pelota se desplaza automáticamente dentro del área de juego. Cuando entra en contacto con una de las paletas o con los bordes superior e inferior de la ventana, cambia su dirección de movimiento simulando un rebote.
Cada vez que la pelota atraviesa uno de los extremos laterales, el jugador contrario obtiene un punto y la pelota regresa automáticamente al centro para continuar la partida. El marcador se actualiza en tiempo real y el juego finaliza cuando uno de los jugadores alcanza cinco puntos, mostrando un mensaje con el ganador.
El programa fue organizado mediante funciones independientes para facilitar su comprensión, mantenimiento y reutilización de código.
